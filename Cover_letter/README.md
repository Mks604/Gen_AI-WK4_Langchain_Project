# 🧠 AI Cover Letter Generator

Generate **professional, ATS-friendly cover letters** instantly using AI.
Built with **Python + Streamlit + OpenAI API**.

---

## 🚀 Demo

A simple and clean web app where users can:

* Enter personal details
* Generate a professional cover letter
* Download it instantly

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/cover-letter-generator.git
cd cover-letter-generator
```

---

### 2️⃣ Create virtual environment (optional)

```bash
python -m venv venv
```

Activate:

```bash
venv\Scripts\activate
```

---

### 3️⃣ Install dependencies

```bash
pip install openai streamlit
```

---

### 4️⃣ Set OpenAI API Key

#### Windows:

```bash
setx OPENAI_API_KEY "your_api_key_here"
```

#### Mac/Linux:

```bash
export OPENAI_API_KEY="your_api_key_here"
```

---

### 5️⃣ Run the application

```bash
streamlit run app.py
```

---

## 🧪 How It Works

1. User enters:

   * Name
   * Job Role
   * Company
   * Skills
   * Experience

2. The app sends this data to OpenAI API

3. AI generates a **professional cover letter**

4. User can:

   * View it
   * Download it
